# Curriculo
Template de currículo feito e pronto para ser editado/utilizado por qualquer um.

Tecnologias

- HTML5
- CSS3
- JavaScript ES6
- PHP 8
- Git/GitHub
- GitHub API

Funcionalidades

- Tema escuro persistente
- Integração com GitHub
- Download de currículo em PDF
- Animações de scroll
- Exibição dinâmica de habilidades
- Renderização dinâmica via PHP
- Layout responsivo
